# 4. 메소드

* 함수와 메소드의 차이점은 this 바인딩 유무 -> 메소드는 this 바인딩이 이루어짐

```
var obj = {
	a: 1,
	b: function bb() {
		console.log(this);
	},
	c: function() {
		console.log(this.a)
	}	
};

obj.b();
obj.c();

console.dir(obj.b);
console.dir(obj.c);
```
